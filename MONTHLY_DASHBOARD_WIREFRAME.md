## Chart Type Guide

```
Bar Chart (Vertical)     → Comparing categories (channels, devices)
Bar Chart (Horizontal)   → Rankings, top N lists (queries, pages)
Grouped Bar             → Current vs previous comparison
Stacked Bar             → Part-of-whole over time
Line Chart              → Trends over time (daily, weekly)
Dual-Axis Line          → Two metrics with different scales
Area Chart (Stacked)    → Composition over time (device split)
Scatter Plot            → Two-variable relationships
Bubble Chart            → Three-variable relationships
Lollipop Chart          → Movers (gainers/losers)
Funnel Chart            → Conversion stages
Histogram               → Distribution (position bands, scores)
Donut Chart             → Part-of-whole (single point in time)
Gauge Chart             → Single metric with thresholds
```
